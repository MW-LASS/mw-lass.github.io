URI Patterns

Basic Pattern Template

The following identifies the elements within a URI and forms the basic pattern using 'data.mw.org.nz':

http://data.mw.org.nz/{type}/{concept} /{reference}


Real-World 'Thing'

http://data.mw.org.nz/id/{concept}/{reference}

/id

 http://data.mw.org.nz/id/cemetery/MtView
Redirects to document http://data.mw.org.nz/doc/cemetery/MtView [303 See Other]
---

Document/Data

http://data.mw.org.nz/doc/{concept}/{reference}

/doc

Vocabularyhttp://data.mw.org.nz/doc/cemetery/MtView >>
>> Content Negotiationhttp://data.mw.org.nz/doc/cemetery/MtView.html [human-readable]
>> Content Negotiationhttp://data.mw.org.nz/doc/cemetery/MtView.rdf [computer-readable]
---

Ontology

http://data.mw.org.nz/def/{concept}/{reference}

/def

Vocabulary http://data.mw.org.nz/def/cemetery
Classhttp://data.mw.org.nz/def/cemetery#Deceased
Propertyhttp://data.mw.org.nz/def/cemetery/cemetery#hasPlot
